Proxy 的缺陷

兼容性

作为 ES6 的特性, Proxy 对于旧版浏览器支持不好, 即使使用 proxy-polyfill 也仅能部分支持 Proxy 的功能

在 github 上的 GoogleChrome/proxy-polyfill 库目前只支持以下 traps:

  • get
  • set
  • apply
  • constructor

性能

Proxy 的性能比 Promise 还差, 也差于 Object.defineProperty
